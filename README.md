<h4 align="center">A clone of the 2048 puzzle game </h4>


## How To Use


```bash
# Clone this repository
$ git clone https://github.com/boya-medellin/2048-game

# Go into the repository
$ cd electron-markdownify

# Make
$ make

# Run the app
$ ./game


```
# Directional movement: 
* h = left
* j = down
* k = up
* l = right
